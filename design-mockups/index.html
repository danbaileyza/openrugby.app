At a glance

How each direction handles the same recurring decisions.

1 · Data Terminal 2 · Editorial 3 · Modern SaaS
Primary nav Top tab bar + ⌘K search Newspaper masthead + underlined tabs Left sidebar with pinned items
Density High — rows, columns, everything tight Low — reads like a weekend paper Medium — cards with room to breathe
Type Inter + JetBrains Mono for numbers Source Serif 4 for display, Inter for body Inter throughout
Background Pure white with grey section fills Warm cream paper (#fbfaf6) Cool zinc-50 with white cards
Accent Deep rugby green Terracotta / FT salmon Indigo / violet gradient
Standings treatment Full stat columns (P, W, D, L, PF, PA, Diff, BP, Pts, Form) Editorial table — serif team names, narrative lede Team logos + trend sparklines per row
Live match Scrolling ticker + compact panel Lead "story" with giant score Featured card with xT + possession
Best for Power users scanning lots of numbers fast Casual fans & editorial / content push Product/app feel, return users with saved views
Laravel lift Low — tightest fit to your current Tailwind stack Medium — adds a web font + layout restructure Medium-high — new sidebar layout + icon set
